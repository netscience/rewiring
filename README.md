# rewiring
Simulador de modelos de reconexión distribuida en redes complejas. Este simulador fue desarrollado en el marco del proyecto de Ciencia de Frontera "Modelos de reconexión para la autoorganización de redes complejas de gran escala (CBF-2025-G-1812)" apoyado por la Secretaría de Ciencia, Humanidades, Tecnología e Innovación (SECIHTI).
